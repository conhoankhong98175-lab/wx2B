# 微信提审材料（可复制模板）

> 所有 `【待填写】` 必须由实际运营者替换。不要把含占位符的材料直接提交审核。

## 基本信息

小程序名称建议：

```text
店告报价助手
```

简介：

```text
面向广告制作、招牌、喷绘写真和图文门店的价格库、报价计算、版本管理与客户确认工具。
```

服务标签建议：

```text
报价单、价格库、门店管理、办公工具、客户确认
```

主类目：

```text
工具 → 报价/比价
```

## 提审功能说明

```text
店告报价助手用于门店内部维护产品价格库、创建项目报价并生成不可变版本。商家可在发布前查看客户视角预览，发布后通过私密链接让客户查看报价、接受、提问或申请修改；商家可复制旧版本生成 V2，旧链接会明确显示状态。系统支持 PDF、分页长图、数据导出和账号删除。不包含支付、订阅、广告、平台水印、定位、手机号快捷登录或用户内容广场。
```

## 审核测试路径

```text
1. 打开小程序后无需账号密码，微信登录会为审核微信创建独立测试商家空间。
2. 首页点击“新建报价”。
3. 填写客户名称，添加任一演示产品，查看本地预估和服务端计算结果。
4. 点击“客户预览并发布”，确认预览后发布 V1。
5. 在报价详情打开客户页，测试“有问题”“申请修改”或“接受报价”。
6. 返回商家端创建新版本并发布 V2，验证 V1 状态被替代。
7. 客户页可打开 PDF；“保存长图”会在用户主动点击后申请相册仅写入权限。
8. 设置页可查看隐私说明、导出数据；删除店铺需输入完整店铺名称二次确认。
```

审核备注：

```text
审核不需要账号密码。小程序通过 wx.login 获取一次性 code，由服务器换取 OpenID 并创建隔离商家空间。演示价格仅用于流程验证，发布前会明确提示核对。客户联系方式为可选字段，公开输出不包含成本、毛利或内部备注。
```

## 隐私保护指引用途文案

相册仅写入：

```text
仅在用户主动点击“保存长图”时，将当前报价长图写入系统相册，用于线下发送或留存；小程序不读取、扫描或上传用户相册中的其他内容。
```

账号标识：

```text
通过微信登录产生的 OpenID 识别商家账号、维持登录状态并隔离不同商家的业务数据，不用于广告画像或向第三方出售。
```

商家和报价数据：

```text
处理商家主动填写的店铺资料、联系人、价格库，以及客户名称、可选联系方式、项目、尺寸、数量、金额、备注和条款，用于计算、保存、展示和追溯报价版本。
```

客户动作：

```text
处理客户主动提交的接受、问题和修改说明，用于通知商家并跟进报价；不要求客户注册账号。
```

用户权利：

```text
商家可在“设置 → 导出数据”获取业务数据，也可在“设置 → 删除店铺”输入完整店铺名称永久删除账号及业务数据。查询、更正或投诉可联系【待填写：隐私联系邮箱】。
```

存储说明：

```text
数据存储在【待填写：服务器所在国家/地区】的受控服务器，业务数据保存【待填写：保存期限或判断规则】；账号删除后立即删除业务数据并使公开链接失效，法律法规要求保留的最小记录除外。
```

## 提审截图清单

- 首页：工作台和“新建报价”入口。
- 价格库：分类、产品和附加项。
- 报价编辑：行项目、金额明细和客户预览按钮。
- 客户预览：不含成本的公开金额和条款。
- 报价详情：V1/V2 状态、客户动作。
- 客户页：PDF、保存长图、接受/修改。
- 设置页：隐私说明、导出和删除店铺。
- 隐私授权：保存长图时的相册仅写入授权说明。
