# 自托管运维说明

## 行业演示库

演示分类、产品公式和附加项维护在 `shared/demo-catalog.json`。它只在新店铺首次初始化时复制一次，更新模板不会修改任何商家已经保存的价格库或历史正式报价。

修改模板后必须执行：

```bash
npm run check
```

演示数据必须保留“演示”和“请核价”提示，不得录入或宣传为统一市场价格。

## 导出与失败重试

- PDF 由服务端按不可变版本实时生成；失败时保留报价版本不变，商家可从版本详情再次点击下载。
- 小程序长图在端内分页生成；失败不会影响正式报价，可按页面提示恢复相册权限后重试。
- 全量 JSON 导出设置了明确的同步规模上限；超限时使用加密数据库备份归档，避免单进程内存耗尽。
- 监控 API 5xx、PDF 生成耗时、磁盘空间和 SQLite WAL；每天执行加密备份，每月实际恢复一份。
